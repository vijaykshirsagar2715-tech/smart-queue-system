# Smart Queue Management System (SQMS)

Smart Queue Management System (SQMS) is a modern web-based application developed using Node.js, TypeScript, and SQL to automate traditional queue handling processes and reduce customer waiting time through real-time queue tracking. The system provides efficient digital token management, seamless queue updates, and smooth user operations with secure backend functionality. REST APIs were developed to ensure effective frontend-backend communication, while optimized database operations improved overall system performance, responsiveness, and usability.
